How to Play Hangman

Welcome to Hangman Bob! This is a game where your goal is to guess the letters for a given blank word on the main screen before Bob is hung. The number of short dashed lines is the length of the correct word. You can guess by either clicking on the letter buttons or using the keyboard keys.

If you guess the incorrect letter it will show up in the left box and Bob will slowly appear. Repeated incorrect or correct letters will not be counted. You have 6 tries to save Bob.
